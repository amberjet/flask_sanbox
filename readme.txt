1. Для запуска приложения установите все требуемые библиотеки из requirements
2. В терминале Pycharm задайте: set FLASK_APP=notes.py
3. Если вы используете Линукс, то не мне вас учить, но замените set на export
4. В том же терминале запустите фласк командой flask run
5. Если вы установили все требуемые библиотеки, всё должно запуститься. Перейдите по ссылке http://127.0.0.1:5000/

О чём приложение:
асоциальное приложение "Заметки"
Никакого добавления в друзьяшки, личных сообщений, возможность читать друг друга и вот этого всего.
Пользователь сурово заходит, сурово регистрируется и сурово пишет заметки только для себя.
Для любого действия нужна регистрация. Только для регистрации регистрация не нужна. Ну, и для залогинивания

С точки зрения хорошего проекта (из статьи на вики) тут есть:
- Регистрация и авторизация пользователей.
Можно зарегистрироваться или войти под существующим профилем.
- Разделение страниц сайта на открытую и закрытую часть.
Поскольку это суровые личные заметки, то открыта только страница регистрации и авторизации
- Базовая система разграничения доступа.
Только зарегистрированный пользователь может просматривать, редактировать и удалять заметки. Причём только свои
(хотя логика построения адресов страниц может вызвать у него коварное стремление влезть в чужие заметки. Но это коварство предусмотрено системой)
- Зарегистрированный пользователь может добавлять, редактировать, удалять сущности системы.
см. пункт выше
- Понятный интерфейс пользователя - использование компонентов Bootstrap
- И есть поиск по словам в заметке - как базовая фильтрация.
К сожалению, разные формы слова не учитываются.
